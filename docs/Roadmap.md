# Roadmap

## Versioning Rule

Roadmap milestones remain three-part versions:

```text
v0.6.1
v0.6.2
v0.6.3
```

Implementation and fix revisions use four-part versions only inside their parent milestone:

```text
v0.6.1.0
v0.6.1.1
v0.6.1.9
```

Fix revisions do not become roadmap milestones.
